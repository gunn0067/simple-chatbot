# 🤖HexaBot - Simple Chatbot (CLI)

A rule-based command-line chatbot built in Python that responds to greetings, FAQs, tells jokes, gives the current time and date, and holds basic conversations.

---

## 📌Description

HexaBot is a simple yet interactive chatbot that runs entirely in the terminal. It uses if-elif logic to match user inputs to predefined responses across multiple categories — greetings, jokes, time/date queries, Python questions and general conversation. Built as part of a Python internship project to practice string handling, conditional logic and building conversational interfaces.

---

## ✨Features

- Responds to greetings and small talk
- Tells 5 random programming jokes
- Displays current time and date
- Answers FAQs about itself
- Responds to Python-related questions
- Help command to guide the user
- Clean exit with farewell message

---

## 🛠️Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core language |

| random | Random joke selection |
